# Rimworld Dead Man Switch - Joint Operation
A mechanoid/weapon expansion mod focused on restoring the ancient joint operation system of ancient combat mechanoid divisions, requiring Dead Man Switch and Dead Man Switch - Ancient Corps as prerequisites.

CE Compatible

Update for 2026-01-15:

1. Added Combat Extended (CE) compatibility:
   - Added CE compatibility for weapons, mechanoids, materials, and apparel.
   - Made slight adjustments to CE's indirect fire mechanics, allowing this mod's artillery mechanoids to simulate the meter-level precision of modern, informationized artillery (they will not suffer severe accuracy penalties due to lack of direct line of sight or forward observers). [Theoretically, a "guided artillery shell" ammunition type should be added, but this was not implemented to reduce logistical (and coding) complexity.]

   - The CE stats for most mechanoids were assigned somewhat arbitrarily and may be slightly rough. Suggestions for adjustments are welcome.
2. Fixed a bug where the Tinker mechanoid could not be resurrected.
3. Added the Mechanoid Relay function to the Quietus Eye mechanoid.
